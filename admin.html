ℹ️ Guide d'utilisation

Utilisez les sections dans le menu de gauche pour modifier le contenu du site :

• Textes du site — Modifiez les titres, descriptions et sous-titres de chaque section.
• Informations contact — Adresse, téléphone, email, horaires.
• Images — Remplacez les photos du héro, galerie, sections, etc.
• Ruban d'annonces — Gérez les messages défilants en bas du site.
• Blog / Actualités — Ajoutez et modifiez les articles.
• Témoignages — Gérez les témoignages affichés sur le site.

Cliquez Enregistrer après chaque modification.
